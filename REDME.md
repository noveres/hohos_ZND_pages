## 1. 卡片大小 → CSS，第 150 行

```css
@media (min-width: 901px) {
    .hoho-carousel {
        --hc-img-max-width: calc(100vw / 5.6);   /* ← 改這裡 */
    }
```
**分母越小 = 卡片越大**（它是「每張卡片佔畫面寬的比例」）：
- `/ 7` → 小（1/7 畫面寬）
- `/ 5.6` → 目前
- `/ 5` → 更大
- `/ 4.5` → 很大

> 卡片是直式 2:3，寬變大高度會自動 ×1.5。

---

## 2. 卡片間距（左右散開）→ JS，第 301 行

```js
desktop: { mode: 'relative', spreadRatio: 0.42, ... }
                                          ↑ 改這裡
```
**`spreadRatio` 越大 = 卡片散越開（間距越大）**：
- `0.35` → 擠
- `0.42` → 目前
- `0.50` → 更開
- `0.60` → 很開

---

## ⚠️ 兩者要「一起調」的關係

卡片**變大**時，若間距沒跟著加大 → 會**重疊**；
卡片**變小**時，間距沒跟著縮 → 中間會**空太開**。

搭配建議（大小 ↔ 間距一起動）：

| 卡片大小 | 建議 spreadRatio |
|---|---|
| `/6.5`（小） | `0.35` |
| `/5.6`（目前） | `0.42` |
| `/5`（大） | `0.48` |
| `/4.5`（很大） | `0.55` |
